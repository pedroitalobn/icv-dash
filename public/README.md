# Logo

O arquivo `logo.png` aqui é um **placeholder** gerado automaticamente.

Para usar o brasão oficial da Cruz da Vida, **substitua** este arquivo:

1. Salve a imagem do brasão como `public/logo.png` (sobrescrevendo este arquivo).
2. Pronto — o cabeçalho do dashboard e a tela de login passam a exibi-lo.

Recomendado: PNG com fundo transparente, ~256x256px.
